# Prism GW Integration Tests

These tests call the real `PrismClient` from the prism-payment package against the gateway.

- Always use `request.json` for the request payload. Never inline payloads in scripts.
- Always use the existing `test.ts` to run tests. If you need different behavior, modify the script.
- If you need a different payload, modify `request.json` before running.
- Run with: `npm run test:checkout-prepare` or `npm run test:payment-profile` from `.tests/prism-gw/`.
- Run all: `npm run test:all`.
